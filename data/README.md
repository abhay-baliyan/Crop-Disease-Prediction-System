# Data Directory

This directory is intentionally left empty in this repository.

The original training, validation, and testing datasets are **not included** because of their large size.

Please download the datasets from the links provided in **dataset_links.md** and organize them according to the project's expected directory structure before training or evaluating the models.

## Supported Crops

- Corn
- Cotton
- Potato
- Rice
- Sugarcane
- Tea
- Wheat

> This repository contains only the source code required to reproduce the project. Dataset ownership and licenses remain with their respective providers.
