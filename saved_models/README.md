# Pre-trained Model Weights

This directory is intentionally left empty in this repository.

The trained model weights (`.pth` files) are not included because they significantly increase the repository size.

## Required Model Files

- corn.pth
- cotton.pth
- potato.pth
- rice.pth
- sugarcane.pth
- tea.pth
- wheat.pth

Place all downloaded model files inside this directory before running the application.

The application automatically loads the appropriate model based on the selected crop.

> Model download instructions will be added in a future release.
